Lista de usuarios

Puede realizar una selección por nombre de usuario (* = comodín), grupo o estatus, y ordenarla por fecha de registro o nombre de usuario, en sentido ascendente o descendente.

Las modificaciones se aplican de una vez a todos los usuarios de los que ha marcado la casilla de verificación (primera columna). Puede

  • eliminar usuarios (el sistema solicita confirmación)
  • cambiar su estatuts
  • incluir o separar de un grupo
  • modificar las propiedades de visualización
  • modificar las preferencias

Si no marca ninguna casilla, las modificaciones se aplican a todos los usuarios en pantalla.
