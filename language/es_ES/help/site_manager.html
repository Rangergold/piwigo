Administración de sitios

Piwigo ofrece la posibilidad de utilizar varios sitios para almacenar los archivos que integran la galería. Esto puede resultarle útil si su galería está instalada en un servidor de alojamiento web con recursos limitados pero necesita mostrar una gran cantidad de archivos.

  1. Edite el archivo tools/create_listing_file.php si necesita modificar determinados parámetros como $conf['prefix_thumbnail'] or $conf['use_exif'].
  2. Ubique el archivo modificado tools/create_listing_file.php en el sitio remoto, en las carpetas definidas con igual nombre que los álbumes para realizar transferencias FTP (como por ejemplo en la carpeta galleries, que es la carpeta general del sitio; de este modo, puede acceder a http://exemple.com/galleries/create_listing_file.php.
  3. Vaya a Administración » Categorías » Admnistración de sitios para crear un nuevo sitio, por ejemplo http://exemple.com/galleries.

  4. Cuando registra un nuevo sitio remoto puede realizar las siguientes Acciones:

    1. generar lista: lanza un comando remoto para generar el listado de archivos
    2. sincronizar: lee el fichero remoto listing.xml y lo sincroniza con la base de datos local
    3. limpiar: elimina el listado remoto
    4. borrar: elimina el sitio y todos los elementos del mismo en base de datos
