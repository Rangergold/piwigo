Miniatyrer skapande

Den här sidan kan du skapa saknas miniatyrer. Det rekommenderas inte att använda den här funktionen, om ditt galleri finns på en delad server.

GD version
GD är bilden manipulera bibliotek för PHP. Välj den version som är installerat på din server. Du kommer att bli varnad av ett felmeddelande om du väljer sort, Gå tillbaka till föregående sida med din webbläsare och välja den andra versionen. Om ingen version som fungerar så stödjer inte din server GD.
Bredd och höjd
Max miniatyrer dimensioner. Original proportionerna bevaras. Standardvärdena kan ändras med hjälp konfigurationsinställningar tn_width och tn_height (se include/config_default.inc.php file).
Antal miniatyrer att skapa
Försök inte att krympa för många bilder på en gång. Miniatyrisering process använder mycket CPU. Om din Piwigo galleri ligger på ett webhotell så kan för hög CPU-last kan leda till indragning av din webbplats.
Filformat
Miniatyrer skapas i JPEG-format. De kan skapas från PNG och JPEG-bilder bara.
